# Shop Backend (Spring Boot)

## Prerequisites
- Java 21+
- Maven
- PostgreSQL

## Database Setup
1. Start PostgreSQL
2. Create database:
   CREATE DATABASE shopdb;
3. Update DB credentials in:
   src/main/resources/application.yml (or application.properties)

## Run Backend
Open terminal in backend folder and run:

mvn spring-boot:run

## Backend URL
http://localhost:8080

## Stop
CTRL + C
